# Awesome Books Rules

## Format

* Dimensions: 150 Ñ… 200 mm
* Bookbinding / cover: V4 (sewn soft binding)

## Local features

Each country has unique local-related design:

* local illustrators
* local design
* local writers
* local curators

## Topics

* History
* Culture
* Food
* Places
* Nature
* Sports
* Industry

## Number of items

* Country: ~100-150
* City: ~30-100
